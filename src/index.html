Початкові кроки

У світі існує безліч різних операційних систем (Windows, Mac OS, Unix-подібні). Програмне забезпечення, написане для однієї системи, не працюватиме на іншій операційній системі. Це пов'язано з різним устроєм виконуваних файлів і функцій операційних систем. Java дозволяє розв'язати проблему запуску однієї програми на різних ОС, пропонуючи користувачам не тільки мову розробки, а й цілу платформу. Основною метою технології java є принцип - «Написане один раз - працює скрізь» (англ. «Write once - run anywhere.» WORA). Ідея полягає в тому, що написавши один раз код програми, можна повторно використовувати його на різних пристроях і операційних системах без будь-яких додаткових зусиль. Важливим завданням java є забезпечення високої ефективності застосунків, що досягається швидким перетворенням байт-коду на мову машинних кодів. Велика увага в java приділяється безпеці. У платформі є безліч класів для шифрування та роботи із сертифікатами. Мова Java замислювався як «мова домогосподарок», тобто вона дуже проста для розуміння та вивчення.
